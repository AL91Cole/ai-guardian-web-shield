# AI Guardian Web Shield Branding Guide

AI Guardian Web Shield should feel calm, sharp, and quietly confident.

The visual identity should say:

- smart protection
- modern trust
- privacy-first care
- simple guidance, not fear

## Recommended Direction

Use **Direction 1: Shield + AI Pulse** as the primary icon system for the current product.

This direction is now the live production identity for the extension.

Why this direction fits best:

- It reads as protection right away
- It feels intelligent without looking robotic
- It stays recognizable at very small browser sizes
- It matches the calm, modern UI already used in the popup and page badge

## Icon Direction 1

### Shield + AI Pulse

Core idea:

- A sleek dark shield symbol inside a rounded square
- One clean pulse line inside the shield
- A small signal node for the "smart protection" feel
- No letters
- Keep it minimal and calm

Meaning:

- shield = protection
- pulse = active, intelligent monitoring
- signal node = smart, modern guidance

### 16x16

- Use a very simple shield outline or filled shield
- Keep the inner pulse to 1 bold zig-zag line only
- Avoid tiny details or thin interior strokes
- The icon should read mostly as a shield first, pulse second

### 32x32

- Show the full shield shape clearly
- Use a slightly thicker pulse line in the center
- Keep the pulse centered and balanced
- Add a soft contrast edge so it stays clear on light and dark browser surfaces

### 128x128

- Use a deeper shield body with a soft gradient from deep teal to dark navy
- Add a mint pulse line with gentle highlight contrast
- Allow a slightly softer inner glow or shadow for polish
- Keep the form minimal and geometric, not glossy

Best use:

- primary browser extension icon
- popup brand mark
- floating page badge mark

## Icon Direction 2

### Shield + Guardian Eye

Core idea:

- A shield shape with a subtle eye form inside
- The eye should feel protective, not watchful or creepy
- The eye can be made from a simple curved negative space and a small center point

Meaning:

- shield = protection
- eye = careful watch

### 16x16

- The eye must become very simple
- Use one shield block with a single eye cutout
- Keep the center detail minimal or remove it if it muddies the shape

### 32x32

- The eye shape can become clearer
- Keep the eye wide and calm, not sharp or aggressive
- Use strong fill contrast so it still reads at toolbar size

### 128x128

- The eye can sit slightly above center inside the shield
- Add gentle curvature so it feels human and calm
- Use mint as a small accent, not the full body color

Best use:

- secondary brand exploration
- trust and guardian storytelling

Watch out for:

- looking like surveillance software
- feeling too intense or suspicious

## Icon Direction 3

### Shield + Circuit Crest

Core idea:

- A shield with 2 or 3 simple circuit paths
- Lines should be minimal and bold
- It should feel intelligent, not busy

Meaning:

- shield = safety
- circuit paths = smart local analysis

### 16x16

- Use only one or two circuit branches
- Make the shield do most of the work
- Skip tiny end nodes if they reduce clarity

### 32x32

- Show a center path with one side branch
- Keep stroke weight strong
- Use round line ends so the icon feels softer and more friendly

### 128x128

- Add 2 to 3 balanced circuit paths
- Use a deep teal shield body with mint connection points
- Keep lots of open space so it still feels premium and minimal

Best use:

- product pages
- future AI or security marketing materials

Watch out for:

- looking too technical
- losing clarity at 16x16

## Color Direction

Use these as the core brand tones:

- `Deep Teal` `#123B52`
- `Dark Navy` `#0B2639`
- `Mint Accent` `#88D7C1`
- `Soft Mint Background` `#DFF7EF`
- `Clean Neutral Background` `#F3F7F7`

Support colors for the product state system can stay softer:

- safe = muted green
- caution = soft olive or amber
- elevated caution = warm amber
- high risk = soft red
- neutral = gray-blue

## Logo Fit With Current UI

The logo should fit the current calm, modern interface like this:

- Use the shield mark on soft neutral cards, not a hard dark square
- Keep the icon body mostly deep teal and dark navy
- Use mint only as the accent line or highlight
- Prefer rounded corners, gentle shadows, and clean spacing
- Avoid glowing cyber effects, loud gradients, or alarm styling
- Avoid using text letters inside the icon

## Small-Size Rules

For browser extension sizes, keep these rules:

- At `16x16`, prioritize silhouette over detail
- At `32x32`, allow one clear inner symbol
- At `128x128`, add polish, not extra complexity
- Never depend on tiny text or tiny letterforms
- Keep the icon readable on both white and mid-tone browser chrome

## Current UI Recommendation

For the product right now:

- Primary icon direction: `Shield + AI Pulse`
- Popup mark: use the shared `Shield + AI Pulse` icon
- Floating page badge mark: use the same shield silhouette and pulse shape
- Toolbar/exported icon set: use the generated icon files in `icons/`
- Keep the icon on a soft neutral field so it stays readable at browser size
